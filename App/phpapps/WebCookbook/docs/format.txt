Why a new file format for recipes?
Every recipe database system belongs on a few fields like title, servings, categories and so on. Then there is the ingredient block in a seperate table and a bigtext field for the preparation. My system has the mandatory fields and the separate ingredient block too. And there are three text fields to separate texts not belonging to the preparation. That is the prefix for descriptions about the recipe, displayed above the ingredient block. Postfix ist the same like prefix, but displayed below the preparation. At last there is the tip field to display special tips for the preparation. If filled, you see the bold Tip: before the text. In addition, all block fields are characterized by a clear introduction 'fieldname:' and a clear end 'fieldname!'. This makes the programming al little easier.

My own format file descrition:

========== Gabel Rezept ==========		// This is the begin of every single recipe
						// Empty
Titel:Serbisches Reisfleisch			// The title of the recipe
Kategorie:112					// The category. You will find a list of sections in /install/xx_tables.sql
Portion:3					// How many portions
						// Empty
Zutaten:					// Begin of the ingredient block
500|g|Schweinefleisch*				// Every line has the three fields amount, value and description. The fields are separated by '|'
2|EL|Sonnenblumenöl
1|Tasse|Reis
||Salz
2||Paprikaschoten, Farbe nach Wunsch
1||Peperoni, Schärfe nach Wunsch
2||Tomaten
2|EL|Tomatenmark
1||große Zwiebel
1||Knoblauchzehe
200|ml|Brühe
||Pfeffer
Zutaten!					// End of the ingredient block
						// Empty
Prefix:						// Begin of the prefix block with free text
Djuvec
Prefix!						// End of the prefix block
						// Empty
Zubereitung:					// Begin of the preparation block with free text.
Reis mit 1 1/2 Tassen Wasser und etwas Salz 10 Min. garen. Falls noch Flüssigkeit im Reis ist, diese abdämpfen, bis der Reis fast trocken ist. Fleisch in gulaschgroße Stücke schneiden, in einem EL Öl braun anbraten. Aus der Pfanne nehmen und beiseite stellen. Paprika putzen und in mundgerechte Stücke schneiden. Peperoni in kleine Stücke schneiden, evtl. entkernen. Tomaten achteln. Zwiebel in grobe Würfel schneiden. Knoblauch hacken. Zwiebel mit dem Tomatenmark im restlichen Öl anschwitzen. Paprika und Peperoni zugeben und ebenfalls etwas anschwitzen. Fleischbrühe angießen, Tomaten, Knoblauch und Salz zugeben, 15 Min. köcheln. Reis zugeben und unter rühren, heiß werden lassen. Zum Schluss kommt das Fleisch wieder rein, alles gut verrühren und kräftig mit Salz und Pfeffer abschmecken.<br><br>
Zubereitung!					// End of the preparation block
						// Empty
Postfix:					// Begin of the postfix block with free text
*Dazu sind alle Kurzbratstücke wie Kamm, Schnitzel oder Rücken geeignet.<br><br>
Postfix!					// End of the postfix block
						// Empty
Tipp:						// Begin of the tip block with free text
Sabberblubber
Tipp!						// End of the tip block
						// Empty
Arbeitszeit: 30 Min.				// Working time (Optional, if no value, suppress line)
Brennwert: 1000					// Calories (Outdatet, do not use! A function is planned, to compute this value from the ingrdient block.)
Image: fleisch/schwein/djuvec.jpg		// Either the image file of the recipe or the term slideshow for more than one image or suppress line
						// Empty
=====						// End of recipe

The recipe should look like follows:

========== Gabel Rezept ==========

Titel:Ananas in Karamellsauce
Kategorie:122
Portion:2

Zutaten:
0.5||Ananas
3|EL|Zucker
100|ml|Fruchtsaft oder
200|ml|Wein
1|Becher|Schlagsahne
1|EL|Puderzucker
||Mark einer halben Vanilleschote
Zutaten!

Zubereitung:
Ananas schälen, vierteln und den Strunk entfernen. Eine Hälfte anderweitig verwenden, die andere in mundgerechte Stücke schneiden. Zucker in einer Kaserolle schmelzen und leicht braun werden lassen. Vom Herd nehmen und vorsichtig die Flüssigkeit zugießen. Achtung, am Anfang verdampft die Flüssigkeit fast explosionsartig, da die Masse ca. 185° heiß wird! Erneut erhitzen und unter Rühren den Zucker schmelzen. Falls Wein verwendet wird, die Flüssigkeit auf die Hälfte reduzieren. Die Ananasstücke zugeben und ca. 10 Min. köcheln. Sahne mit Puderzucker und Vanillemark steif schlagen. Ananas auskühlen lassen, auf zwei Portionsschälchen verteilen und die Sahne darüber geben.<br><br>
Zubereitung!

Postfix:
Als Saft ist Orangensaft die erste Wahl, beim Wein Weiß oder Rot nach Geschmack.<br><br>Die Ananas ist nach einer Idee von <b>Johann Lafer</b>. Der brachte die aber in einem Schaumomelette unter.<br><br>
Postfix!

Wozu ein neues Format für den Rezeptaustausch?
Jede Rezeptanwendung hat die obligatorischen Felder wie Titel, Anzahl der Portionen, Kategorien etc. Dann folgt der Zutatenblock, der üblicherweise in eine separate Tabelle ausgelagert ist. Zuletzt kommt ein großes Textfeld für die Zubereitungsanleitung. Darein kommen i.d.R. auch alle Texte, die eigentlich in der Zubereitung nichts zu suchen haben. Ich habe nun dieses Feld in vier Felder aufgeteilt: Prefix enthält Erläuterungen, die vor dem Zutatenblock dargestellt werden. Preparation ist für die eigentliche Zubereitungsanleitung. Postfix ist ähnlich Prefix, wird aber nach der Zubereitung dargestellt. Danach folgt noch das Tipp Feld, in das Erläuterungen zu speziellen Zubereitungen fließen sollen. Erkennbar an dem Fett geschriebenen Tipp:. Außerdem sind alle Blockfelder durch eine klare Einleitung 'feldname:' und ein klares Ende 'feldname!' gekennzeichnet. Das erleichtert etwas die Erfassung per Programm.

Die Beschreibung meines eigenen Übertragungsformats:

========== Gabel Rezept ==========		// Der Beginn jedes einzelnen Rezepts
						// Leer
Titel:Serbisches Reisfleisch			// Der Titel
Kategorie:112					// Die Kategorie. Eine Liste findet sich in /install/xx_tables.sql
Portion:3					// Anzahl der Portionen
						// Leer
Zutaten:					// Beginn des Zutaten Blocks
500|g|Schweinefleisch*				// Jede Zeile hat die drei Felder Menge, Wert und Beschreibung. Die Felder sind mit einem '|' getrennt.

200|ml|Brühe

						// Leer
Prefix:						// Beginn des Prefix blocks mit freiem Text
Djuvec
Prefix!						// Ende des Prefix Blocks
						// Leer
Zubereitung:					// Beginn des Zubereitungsblocks mit freiem Text

Zubereitung!					// Ende des Zutatenblocks
						// Leer
Postfix:					// Beginn des Postfix Blocks mit freiem Text
*Dazu sind alle Kurzbratstücke wie Kamm, Schnitzel oder Rücken geeignet.<br><br>
Postfix!					// Ende des Postfix Blocks
						// Leer
Tipp:						// Beginn des Tipp Blocks mit freiem Text
Sabberblubber
Tipp!						// Ende des Tipp Blocks
						// Leer
Arbeitszeit: 30 Min.				// Optional, falls kein Wert vorhanden, Zeile weg lassen

						// Leer
